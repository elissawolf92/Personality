User Study for Character Animation with Personality 

This study aims to find a mapping between motion parameters and personality types. It asks you to compare two characters with different motion styles  and to answer two independent questions per motion style.

 

Instructions

  1. Please install the Unity Web Player from the link provided below.

  2. Please select a unique username. You can run the study with this username in multiple sessions. If this is your first session and you get a warning message that the username already exists, please enter another name.

  3. Please select your gender and write your age in the text box.

  4. Press the "Start" button.

  5. There are two animations demonstrating the character performing different actions each with different motion styles.

  6. Answer the  items asking you to compare the two characters for each setting and press the "Submit" button.  These items inquire which character you think exhibits certain personality traits better compared to the other.

  7. After selecting all the answers and pressing the "Submit" button, you will move on to the next question. If you press "Submit" before selecting all the items you will get a warning message.

  8.  If you want to go to the previous or next questions without submitting, you can use the left and right arrows on your keyboard.

  9. The study is over when you finish answering all the 96 questions (2 animations x 48 motion styles).
     

Controls

Keyboard:

  • Left and right keys: Change motion style

  • Up and down keys: Change animation
